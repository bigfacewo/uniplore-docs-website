Network File

组件介绍​

“加载网络文件”（Network File）控件用于读取指定格式的图文件。

  • 输入：
    • 无
  • 输出：
    • data：图文件属性
    • nwdt：网络图实例

“加载网络文件”（Network File）控件能够读取的文件类型包括.net，.gml 和 edgelists 等格式。

页面介绍​

点击“加载网络文件”（Network File）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
图信息加载图文件并且根据图构建数据表
点数据信息图中节点的信息。读取标准数据文件
基本信息显示图的类型、点的个数、边的个数
