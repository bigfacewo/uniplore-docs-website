Corpus

组件介绍​

“加载语料库”（Corpus）加载文本文档，（可选）用类别标记。

  • 输入：
    • 无
  • 输出：
    • cor：文本实例

页面介绍​

点击“加载语料库”（Corpus）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
标签选项样例数据：系统提供的样例文本数据集，可直接选择加载
手动上传：上传自定义文本数据集
标题配置文本数据集的标题属性
可选属性/忽略属性配置文本数据集的可选属性以及忽略属性
