Random Data

组件介绍​

“数据生成”（Random Data）控件根据用户配置生成数据集。

  • 输入：
    • 无
  • 输出：
    • data：数据集

页面介绍​

点击“数据生成”（Random Data）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
生成方法列表指定生成规则，并配置规则信息，方便生成相应的数据
数据规则数据规则选项卡（包括：正态分布、伯努利分布、二项分布、连续型均与分布、离散型均匀分布、多项分布、超几何分布、负二项分布、泊松分布、指数分布、伽玛分布、Student-T 分布、二维正态分布）
生成实例数需要生成的数据实例数
