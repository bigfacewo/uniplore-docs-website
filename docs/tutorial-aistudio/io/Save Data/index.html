Save Data

组件介绍​

“保存数据”（Save Data）控件将处理后的数据集保存为文件。

  • 输入：
    • data：数据集
  • 输出：
    • 无

页面介绍​

点击“保存数据”（Save Data）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
文件名需要保存的文件名称mydata
保存类型可以选择三种类型：.csv/.tab/.pickle.csv

使用案例​

在下图所示的案例中，使用“加载文件”（save-data）控件加载样例数据集，接着连接“保存数据”（Save Data）控件将样例数据集保存为文件。
