Predictions

组件介绍​

“预测”（Predictions）控件显示模型对数据的预测。该控件需要输入一个数据集以及一个或多个预测器（包括分类器、非学习算法等），然后输出评估结果以及包含预测结果的原始数据集。

  • 输入：
    • data：训练数据集
    • mod：要在数据上使用的预测器
  • 输出：
    • evr：评估结果
    • prdt：增加了预测的原始数据集
