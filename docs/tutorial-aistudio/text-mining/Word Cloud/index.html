Word Cloud

组件介绍​

“词云（Word Cloud）”控件根据输入的文本数据集生成词云，词云中词汇的大小表示了表示词汇出现的频率。

  • 输入：
    • cor：文本实例
    • top：选择的主题
  • 输出：
    • cor：文本实例
    • sel：选中的词
    • wdc：词表

页面介绍​

点击查看词云按钮，查看绘制的词云：

参数选项​

选项说明样例值
添加颜色是否对生成的词云进行颜色标记
单词角度词云中单词的角度
词云表文本中出现的单词以及对应的权重
