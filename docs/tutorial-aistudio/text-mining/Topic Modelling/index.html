Topic Modelling

组件介绍​

“主题模型（Topic Modelling）”控件用于对输入的文本数据集进行情感分析。

  • 输入：
    • cor：文本数据集
  • 输出：
    • cor：文本数据集
    • selTop：选择主题的单词权重
    • allTop：所有关键字的权重
