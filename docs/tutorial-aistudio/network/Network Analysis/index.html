Network Analysis

组件介绍​

网络分析器（Network Analysis）控件用于对图数据进行统计分析。

  • 输入：
    • nwdt：网络图实例
  • 输出：
    • data：图文件属性
    • nwdt：网络图实例

页面介绍​

点击网络分析器（Network Analysis）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
图级别目录对网络图中图级别信息进行统计分析
点级别目录对网络图中点级别信息进行统计分析

使用案例​

在下图所示的案例中，使用网络生成器（Network Generator）控件生成网络图，之后连接网络分析器（Network Analysis）控件对图数据进行统计分析。

案例中全部组件使用默认配置，网络分析器（Network Analysis）中勾选图级别目录中的节点数目、边数目、平均度、密度、直径、半径，其对应的数目将在后面展示。案例中组件的配置以及执行结果如下图所示。
