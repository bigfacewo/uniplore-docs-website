Tree Viewer

组件介绍​

“决策树查看器（Tree Viewer）”控件用于对分类多变量数据进行对应分析。Correspondence Analysis（CA）根据输入数据集，计算其对应的线性变换。和 PCA 相似，CA 被用来计算离散数据而非连续数据的线性变换。

  • 输入：
    • data：数据集
  • 输出：
    • 无

参数选项​

选项说明样例值
节点节点个数
叶子叶子节点个数
深度树显示的深度
边宽树的边宽
  固定宽度：所有边的宽度相同
  相对宽度（根）：边的宽度根据节点中实例相对于训练数据中的所有实例的比列进行变化
  相对宽度（父）：边的宽度根据节点中实例相对于父节点中的实例的比例进行变化
目标类树中只展示该值相关信息
