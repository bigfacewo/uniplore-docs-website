CN2 Rule Viewer

组件介绍​

“CN2规则查看器（CN2 Rule Viewer）”控件用于查看已训练的CN2模型的规则信息。

  • 输入：
    • data：数据集
    • mod: CN2规则分类器，包括诱导规则列表
  • 输出：
    • sel: 选择规则对应的数据
    • data: 数据集
