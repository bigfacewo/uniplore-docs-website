Venn Diagram

组件介绍​

“维恩图（Venn Diagram）”控件用于绘制数据子集的维恩图。

  • 输入：
    • data: 数据集
  • 输出：
    • 无

页面介绍​

点击“维恩图（Venn Diagram）”控件查看参数配置页面，如下图所示：

点击“查看结果”按钮，查看数据集详情：

参数选项​

选项说明样例值
信息输入数据集信息
数据实例标识用于比较数据的标识符

使用案例​

在下图所示的案例中，使用“加载文件（File）”控件加载数据集，连接“数据抽样（Data Sampler）”进行抽样，接着使用“维恩图（Venn Diagram）”控件连接“加载文件（File）”和“数据抽样（Data Sampler）”的输出绘制韦恩图。
