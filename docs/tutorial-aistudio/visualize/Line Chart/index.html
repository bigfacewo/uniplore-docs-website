Line Chart

组件介绍​

“线形图表（Line Chart）”控件通过线性图表的方式可视化时序数据，同时可以接收时序模型预测结果，展示预测信息。

  • 输入：
    • tsd：时序数据
    • fore：时序模型的预测结果
  • 输出：
    • 无

选项说明样例值
图表类型图表的展示类型，包括：折线图、阶梯线图、柱状图、面积图、光滑样条图
属性列表选择需要绘制线形图表的属性，可多选
