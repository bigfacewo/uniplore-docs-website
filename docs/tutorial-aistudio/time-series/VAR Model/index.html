VAR Model

组件介绍​

“VAR模型”（VAR Model）控件根据输入数据集，以及配置参数进行构建VAR时序预测模型。

  • 输入：
    • tsd：时序数据
  • 输出：
    • tsm：时序预测模型
    • fore：预测的时序数据
    • fit：模型实际拟合的值，等于原始值减去残差
    • resid：模型在每一步的预测误差
