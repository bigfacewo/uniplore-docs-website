Interpolate

组件介绍​

“插值处理”（Interpolate）控件对时序数据中的缺失值进行插值处理。

  • 输入：
    • tsd：时序数据
  • 输出：
    • tsd：处理后的数据

页面介绍​

点击“插值处理”（Interpolate）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
缺失值插补方案处理缺失值的方法：
  线性插值法（Linear）
  三次样条插值法（Spline）
  最近邻插值法（Nearest）
  平均值插值法（Mean）
多元多项式插值多元多项式处理

使用案例​

在下图所示的案例中，使用“加载文件”（File）控件加载数据，使用“查看数据”（Data Table）查看数据集，同时连接“插值处理”（Interpolate）控件对数据进行插值处理，之后使用“查看数据”（Data Table）控件查看处理后的数据。
