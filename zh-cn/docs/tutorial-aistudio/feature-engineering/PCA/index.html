PCA

组件介绍​

“主成分分析”（PCA）控件将主成分分析方法应用于输入数据。控件对输入数据计算其 PCA 线性转换，输出经过转换的带权重的数据或特征向量。

  • 输入：
    • data：数据集
  • 输出：
    • data：经过PCA变换后的数据
    • cmp：特征向量
