Corpus Viewer

组件介绍​

“语料库查看器（Corpus Viewer）”控件用于查看输入的文本数据集。

  • 输入：
    • cor：文本数据集
  • 输出：
    • cor：文本数据集

页面介绍​

点击“语料库查看器（Corpus Viewer）”控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
基本信息文档：输入的文档数量
预处理：是否使用预处理器
POS标签：结果为真或假
N-grams 范围：如果 N-grams 在预处理文本中设置，结果会显示，默认为 1-1
匹配数：RegExpi 过滤的文档数量
搜索特征RegExp 筛选器过滤特性
显示特征显示特征
RegExp过滤器用于过滤文档的 Python 正则表达式
