Random Forest Viewer

组件介绍​

“随机森林查看器（Random Forest Viewer）”控件可用于可视化随机森林模型。

  • 输入：
    • mod：随机森林模型
  • 输出：
    • mod：选定的树模型

页面介绍​

点击“随机森林查看器（Random Forest Viewer）”控件查看参数配置页面，如下图所示：

点击“查看结果”按钮，查看数据集详情：

参数选项​

选项说明样例值
森林信息显示随机森林中树的数量
树深度可视化图中显示的每棵树的深度
