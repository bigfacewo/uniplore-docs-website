Distributions

组件介绍​

“数据分布（Distributions）”控件用于显示单个属性值的分布。对于离散属性，“Distributions”组件显示每个属性值出现在数据集中的次数。

  • 输入：
    • data: 数据集
  • 输出：
    • 无

页面介绍​

选项说明样例值
特征变量分布显示列表
分箱宽度对于连续型属性划分的区间个数
分组属性选择分组按类属性显示
