Naive Bayes

组件介绍​

“朴素贝叶斯（Naive Bayes）”控件是基于贝叶斯定理和特征独立假设的快速简单概率分类器。朴素贝叶斯从数据中学习一个朴素贝叶斯模型，它只适用于分类任务。

  • 输入：
    • data：数据集
    • pre: 预处理方法
  • 输出：
    • lrn: 在对话框中配置参数后的朴素贝叶斯学习算法
    • mod: 已训练的模型（仅当输入端data存在时，才会有输出信息）
