Constant

组件介绍​

“一致性算法（Constant）”控件预测训练集中最频繁的类或均值。

  • 输入：
    • data：数据集
    • pre: 预处理方法
  • 输出：
    • lrn: 多数/平均学习算法
    • mod: 已训练的模型（仅当输入端data存在时，才会有输出信息）
