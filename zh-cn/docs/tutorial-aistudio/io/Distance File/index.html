Distance File

组件介绍​

“加载距离文件”（Distance File）控件用于加载已有的距离文件。

  • 输入：
    • 无
  • 输出：
    • dis：距离矩阵

页面介绍​

参数选项​

选项说明样例值
标签选项样例数据：系统提供的样例数据，可直接选择加载
手动上传：上传自定义距离矩阵
基本信息显示距离矩阵相关信息
