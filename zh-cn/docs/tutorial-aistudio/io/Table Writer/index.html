Table Writer

组件介绍​

“数据入库”（Table Writer）控件支持把数据集写入指定数据库中。

  • 输入：
    • data：数据集
  • 输出：
    • 无

页面介绍​

点击“数据入库”（Table Writer）控件查看参数配置页面，如下图所示：

参数选项​

选项说明样例值
数据库类型连接数据库的类型（Microsoft SQL Server, PostgreSQL, MySQL, Oracle）PostgreSQL
数据库地址数据库连接地址127.0.0.1
端口连接端口号5432
数据库名数据库名test
用户名数据库连接用户名postgres
密码数据库连接密码123456

配置完成后，点击“获取表信息”按钮，将会弹出“数据入库”（Table Writer）控件的交互页面（如下图所示）。在交互页面中可以选择新建数据表把数据写入库中，或者选择已有表进行追加写入或覆盖写入。
