ARIMA Model

组件介绍​

“ARIMA模型”（ARIMA Model）控件根据输入数据集，以及配置参数进行构建ARMA，ARIMA或ARIMAX时序预测模型。

  • 输入：
    • tsd：时序数据
    • exog：可用于 ARIMAX 模型的附加自变量时序数据
  • 输出：
    • tsm：时序预测模型
    • fore：预测的时序数据
    • fit：模型实际拟合的值，等于原始值减去残差
    • resid：模型在每一步的预测误差

参数选项​

选项说明样例值

差分阶数（d）
滑动平均项数（q）

预测期数模型预测多少期数据
置信区间指定预测结果的置信区间
